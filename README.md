# 🎯 Placement Mentor Lite

> An AI-powered career readiness dashboard for AI & Data Science students — built with Python, Scikit-Learn, and Streamlit.

Placement Mentor Lite helps students understand exactly where they stand before placements, what is missing from their profile, and what to do about it — with a personalized 30/60/90-day roadmap, ML-based placement probability, and visual gap analysis.

## 📊 Dataset

The training dataset (`data/placement_sample.csv`) is a **synthetic** dataset of 200 student records designed to teach the ML model from feature *combinations*, not single factors alone.

### Distribution

| Group | Count | Placed % | Description |
|---|---|---|---|
| Excellent | 40 | ~90 % | High CGPA, skills, projects, internship |
| Good | 50 | ~70 % | Above-average in most areas |
| Average | 50 | ~45 % | Mid-range profile |
| Weak | 40 | ~15 % | Below-average across most areas |
| Borderline | 20 | ~50 % | Strong in one area, weak in another |

### Realistic Exceptions (intentional)

- High CGPA but not placed → because coding activity was very low
- Low CGPA but placed → because of strong coding + projects combination
- Internship completed but poor coding → mixed outcome
- Excellent coding activity but no internship → borderline case

These exceptions force the model to learn from combinations rather than memorising one feature.

### Columns

| Column | Type | Description |
|---|---|---|
| `cgpa` | float | GPA out of 10.0 |
| `projects` | int | Number of completed projects |
| `coding_platform_activity` | int | Problems solved on coding platforms |
| `internship_score` | float | 0.0 / 0.3 / 0.7 / 1.0 |
| `skill_match_percentage` | float | % of target-role skills matched |
| `placed` | int | Target label: 1 = placed, 0 = not placed |

To regenerate the dataset:

```bash
python scripts/generate_dataset.py
```

## 🤖 Machine Learning Pipeline

### Step 1 — Generate Dataset

```bash
python scripts/generate_dataset.py
```

Creates `data/placement_sample.csv` with 200 balanced synthetic records.

### Step 2 — Train and Evaluate

```bash
python scripts/train_model.py
```

Trains all three models, prints comparison table, saves best model and evaluation charts.

### Step 3 — Run Dashboard

```bash
streamlit run app.py
```

The dashboard loads the saved `.pkl` model automatically — no retraining needed.

## 📈 Model Comparison

| Model | Accuracy | Notes |

## 🏅 Readiness Grade Scale

| Score | Grade |
|---|---|
| 90 – 100 | 🟢 Excellent |
| 75 – 89 | 🔵 Good |
| 60 – 74 | 🟡 Average |
| Below 60 | 🔴 Needs Improvement |

### Score Breakdown (out of 100)

| Component | Max Points | Based On |
|---|---|---|
| CGPA | 20 | `(cgpa / 10) × 20` |
| Skills | 25 | Role skill match percentage |
| Projects | 20 | `(projects / 4) × 20` |
| Internship | 10 | Status level (0 / 4 / 8 / 10) |
| Coding Practice | 25 | `(activity / 200) × 25` |

## 🎤 Interview Pitch (60 seconds)

> "Placement Mentor Lite is a Python and Streamlit dashboard that measures placement readiness for AI and Data Science students. It takes inputs like CGPA, skills, projects, internship status, coding activity, and target role. It then runs three ML models — Logistic Regression, Decision Tree, and Random Forest — compares them on accuracy, precision, recall, and F1 score, picks the best one, and shows feature importance from Random Forest. It calculates a readiness score out of 100 with a grade label, and generates a personalized 30/60/90-day roadmap based on the student's exact skill gaps. All profiles are saved to CSV using Pandas. I also built a separate training script that generates evaluation charts including a confusion matrix and model comparison chart."
